## Usage Guidelines

### Progress Rings
- Use **Primary** color for active goals
- Use **Success** color when goal is 80%+ complete
- Use **Warning** color when deadline is within 7 days

### Goal Cards
- Background: `card` color
- Border: `border` color
- Category badge: Use category-specific color

### Buttons
- Primary actions: `primary` background
- Secondary actions: `secondary` background
- Destructive actions: `destructive` background

### AI Coaching Panel
- Use `info` (violet) color for AI-related elements
- Streaming text: `foreground` color
- Insights badge: `info` background

## Accessibility Notes

All color combinations meet **WCAG 2.1 AA** standards:
- Normal text: 4.5:1 contrast ratio minimum
- Large text: 3:1 contrast ratio minimum
- UI components: 3:1 contrast ratio minimum

Test all colors with:
```bash
# Install contrast checker
npm install -g wcag-contrast

# Test combinations
wcag-contrast "#3B82F6" "#FFFFFF"  # Primary on white
wcag-contrast "#60A5FA" "#000000"  # Primary on black
```
